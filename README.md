# `nns`

## Overview
This repo includes various programs relating to *n*eural *n*etwork*s*, hence `nns`. Some programs are implemented with C and some in Python. For notes and explanations about nerual nets and AI in general, refer to [this](https://github.com/ngivanyh/cs-notes/tree/main/AI).
